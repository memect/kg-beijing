## Introduction

- Javascript Object Notation
- 优点
    - 易于读写
    - 易于机器解析和生成
- 构建与两种结构
    - name/value pairs. 可以理解为Python中的`dict`. 也可以成为`对象`
        - Like: {"a":1, "b":2} 
    - ordered list of values. 可以理解为`Array`.
        - Like: ["a", "b", 1, 2]
    - value: 可以是**双引号**括起来的字符串（string）、数值(number)、true、false、 null、对象（object）或者数组（array).
